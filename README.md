
# 🖥️ Repositório de Estudos: Sistemas Operacionais

Bem-vindo ao meu repositório de estudos sobre **Sistemas Operacionais**. Este espaço foi criado para consolidar e organizar os conhecimentos fundamentais adquiridos ao longo da disciplina, abrangendo desde a evolução histórica do hardware até as técnicas modernas de virtualização e containerização.

## 📌 Objetivo

O objetivo deste repositório é servir como um guia de referência rápida e base de conhecimento técnica. O conteúdo está estruturado em arquivos Markdown (`.md`) para facilitar a leitura diretamente no GitHub, utilizando uma linguagem clara e diagramas conceituais.

-----

## 🗂️ Conteúdos do Repositório

Os materiais estão divididos nos seguintes tópicos principais:

### 1\. [Fundamentos e Evolução Histórica](https://www.google.com/search?q=./resumos/evolucao-historica.md)

  * As cinco gerações da computação (Válvulas à Mobilidade).
  * O surgimento do UNIX e seu impacto nos sistemas modernos.
  * Conceitos de Multiprogramação e Time-sharing.

### 2\. [Estrutura e Arquitetura de Sistemas](https://www.google.com/search?q=./resumos/estrutura-arquitetura.md)

  * **Kernel:** O núcleo do sistema e suas responsabilidades.
  * **Modo Usuário vs. Modo Kernel:** Camadas de proteção e privilégios.
  * **Modelos de Kernel:** Monolítico, Microkernel e Híbrido.
  * **System Calls:** A interface entre aplicações e o hardware.

### 3\. [Gerenciamento de Processos](https://www.google.com/search?q=./resumos/processos.md)

  * Definição de Processo (Programa em execução).
  * Estados do Processo (Novo, Pronto, Executando, Bloqueado, Finalizado).
  * **PCB (Process Control Block)** e Troca de Contexto (*Context Switch*).
  * Hierarquia de Processos (Pai e Filho).

### 4\. [Gerenciamento de Memória](https://www.google.com/search?q=./resumos/memoria.md)

  * Hierarquia de Memória (Registradores, Cache, RAM e Disco).
  * **Paginação e Segmentação:** Técnicas para evitar a fragmentação.
  * **Memória Virtual:** Execução de programas maiores que a RAM física.

### 5\. [Virtualização e Tecnologias Modernas](https://www.google.com/search?q=./resumos/virtualizacao.md)

  * Introdução a Hypervisors e Máquinas Virtuais (VMs).
  * Uso do Oracle VirtualBox para isolamento de ambientes.
  * **Estudo de Caso:** Transição de infraestruturas locais para Nuvem (AWS) e Containers (Docker).

-----

## 🚀 Como utilizar este repositório

Cada pasta ou arquivo `.md` contém um resumo focado em um tema específico. Recomenda-se seguir a ordem numérica para uma compreensão progressiva da complexidade dos sistemas operacionais.

  * **Sistemas:** Windows, Linux (Ubuntu/Lubuntu), UNIX.
  * **Virtualização:** Oracle VirtualBox.
  * **Containers:** Docker.
  * **Controle de Versão:** Git e GitHub.
